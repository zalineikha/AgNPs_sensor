# AgNPs_LC sensor for cadmium detection

This is the ML repository for "A Paper-Based Colorimetric Sensor Integrating DFT Insights and Machine Learning for Cadmium Detection in Water".

Contents:
1. Image dataset of AgNPs-LC as "AgNPs_LC" folder.
2. Image dataset of Cd2+-bridged AgNPs-LC as "AgNPs_LC-Cd" folder.
3. Python script for ML implementation as "Cd_detection" file.
